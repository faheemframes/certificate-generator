# Certificate Generator 

A simple, offline certificate generator tool. Upload your template, enter names, and generate personalized certificates instantly!

##  Features
- Upload any certificate template (PNG/JPG)
- Real-time preview as you adjust settings
- Adjust position, font size, and color with sliders
- Download individual certificates
- Download all certificates as ZIP

## How to Use
1. Open the tool: [Live Demo](https://faheemframes.github.io/certificate-generator/index.html)
2. Upload your certificate template (PNG/JPG)
3. Enter names (one per line)
4. Adjust text position, size, and color using sliders
5. Click "Generate All Certificates"
6. Download individual certificates OR click "Download All as ZIP"
