# certificate-generator
Certificate generator tool

A simple, offline certificate generator tool. Upload your template, enter names, and generate personalized certificates instantly!

##  Features
- Upload any certificate template (PNG/JPG)
- Real-time preview
- Adjust position, font size, and color with sliders
- Download individual certificates
- Download all as ZIP

## How to Use
1. Open the tool: [Live Demo](https://faheemframes.github.io/certificate-generator/certificate_generator_realtime.html)
2. Upload your certificate template
3. Enter names (one per line)
4. Adjust position using sliders
5. Generate and download!

## Usage
- Works completely offline
- No installation required
- No data uploaded to servers
- 100% client-side
